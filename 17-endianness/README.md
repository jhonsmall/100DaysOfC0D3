
## Equivalencias
```
4  BITS    = NIBBLE =      1010 =                0xA
2  NIBBLES = BYTE   = 1001-1010 =               0x9A
16 BITS    = WORD   =  2 BYTES  =             0xFD9A
32 BITS    = DWORD  =  4 BYTES  =         0xCD01FD9A
64 BITS    = QWORD  =  8 BYTES  = 0x5D9C00F3CD01FD9A
```
